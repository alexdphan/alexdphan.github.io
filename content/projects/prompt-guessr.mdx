Prompt Guessr a cool word-guessing game.

You get a prompt and have to come up with a word that matches it.

Then the game checks it against a list of correct answers and gives you a score. You can play it alone or with a group of friends.

The best part? The code is open-source, so anyone can modify it and make their own version. Plus, it's super easy to set up and run on your computer.

Overall, if you're looking for a fun and customizable game, check out Promptguessr!
